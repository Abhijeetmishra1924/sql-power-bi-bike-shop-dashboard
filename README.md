# sql-power-bi-bike-shop-dashboard
Project Overview

This project focuses on building an end-to-end Sales Analysis Dashboard for a Bike Shop using SQL for data processing and Power BI for visualization.
The goal is to transform raw transactional data into meaningful insights that help improve sales performance, inventory planning, and customer analysis.
